# TRAVI Page CRO Analysis

_Created: 2026-01-28_
_Goal: Identify conversion improvements aligned with information-first strategy_

---

## Current Conversion Goals (Phase 1: Information Authority)

**Primary Goals:**

1. Page views (content consumption)
2. Return visits (trust building)
3. Affiliate clicks (ticket bookings via Tiqets)

**NOT optimizing for (yet):**

- Email signups (avoid aggressive capture)
- App downloads
- Trip planning CTAs

---

## Homepage Analysis

### Above the Fold Assessment

**Current State:**

- Hero with "Know Before You Go" headline
- Search bar
- Destination carousel
- Stats (17 destinations, 3,000+ attractions)

**CRO Recommendations:**

#### Quick Wins

1. ✅ **Headline improved** - Changed from generic "Your Trusted Travel Resource" to benefit-focused "Know Before You Go"
2. ✅ **Badge clarity** - Now shows concrete "17 Destinations · 3,000+ Attractions" instead of vague tagline
3. ✅ **CTA improved** - Changed from "Start Exploring" to action-oriented "Explore Destinations"

#### High-Impact Changes Needed

1. **Search bar prominence** - Make search more prominent with popular searches visible
2. **Trust signals** - Add "Updated daily" or "Free to use" near hero
3. **Clear value prop** - Show what makes TRAVI different (detailed info, not reviews)

### Content Sections

**Destinations Grid:**

- Good: Visual cards with destination names
- Improve: Add attraction count per destination ("150+ things to do")
- Improve: Show "Recently updated" badge for fresh content

**FAQ Section:**

- ✅ Updated to be more conversational and answer real questions
- Good: Addresses "Is it free?" objection
- Good: Explains what kind of information TRAVI provides

## Attraction Detail Pages (Key Conversion Pages)

### Current Structure

- Hero with attraction image
- Key facts (hours, price, duration)
- Description
- Tickets section (affiliate)
- FAQ

### CRO Recommendations

#### For Information Goal

1. **Comprehensive info first** - Ensure hours, prices, best times are prominent
2. **Add "How to get there"** section with transport options
3. **Add "Nearby attractions"** for internal linking
4. **Add "Best time to visit"** section

#### For Affiliate Goal

1. **Ticket CTA visibility** - Ensure "Book Tickets" button is visible without scroll
2. **Skip-the-line benefit** - Highlight time saved
3. **Price transparency** - Show "From $XX" pricing
4. **Trust signals** - "Free cancellation" and "Mobile tickets" badges

## Trust Signal Audit

### Current Trust Signals

- Stats (destinations, attractions count)
- "Updated daily" messaging
- Clean, professional design

### Recommended Additions

1. "Last updated: [date]" on attraction pages
2. "Data verified from official sources" statement
3. "Free to use, no signup required" messaging
4. Source attribution for opening hours/prices

## Internal Linking Strategy

### Current State

- Basic navigation
- Breadcrumbs on some pages

### Recommendations

1. **Related attractions** - "If you liked X, also consider Y"
2. **Category links** - "More museums in Paris"
3. **Destination links** - Every attraction → parent city
4. **Cross-sell** - "Looking for hotels nearby?"
