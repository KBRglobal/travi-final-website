PHASE 1 — SEO & INDEX CLEANUP
================================

1. FULL SITE AUDIT
Scan the ENTIRE codebase and routing system.
Identify ALL public routes.

For EACH route:
- Decide: KEEP INDEXED or REMOVE FROM INDEX
- If content is incomplete, generic, lorem, marketing, demo, or speculative → NOINDEX + NOFOLLOW

MANDATORY NOINDEX:
- /coming-soon (keep page, but fully blocked)
- Any page with countdown, “launch”, “notify me”
- Any page not serving end-user value TODAY

DELETE OR NOINDEX:
- Any unused language routes
- Any abandoned experiment pages
- Any routes not linked from main navigation

2. CANONICAL ENFORCEMENT
For EVERY public page:
- Exactly ONE canonical URL
- Canonical must point to EN version
- No self-conflicting canonicals
- No missing canonicals

3. META TAG CORRECTION
For EVERY indexed page:
- Unique <title>
- Unique meta description
- No generic branding titles (e.g. “Expert Travel Guides – TRAVI”)
- Titles must describe the PAGE, not the SITE

If Google is likely to rewrite → FIX IT.

4. SITEMAP
- Sitemap must include ONLY indexable pages
- Remove any noindex, admin, placeholder, or blocked routes
- Validate sitemap completeness

5. ROBOTS.TXT
- Explicitly disallow:
  /coming-soon
  /login
  /access-denied
  /test
  any inactive locale paths
- Ensure sitemap is declared
