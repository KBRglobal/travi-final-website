## Reporting a Vulnerability

We take security seriously and appreciate responsible disclosure.

If you discover a security vulnerability, please **do not open a public GitHub issue**.

Instead, report it privately by emailing:

üìß **security@travi.world**

Please include:
- A clear description of the issue  
- Steps to reproduce (if applicable)  
- Potential impact  
- Any relevant screenshots or logs  

### What to expect
- Initial response within **72 hours**
- Status update after review
- Coordinated disclosure once resolved, if applicable

We do not currently offer a public bug bounty program, but we acknowledge and appreciate security researchers who report issues responsibly.
